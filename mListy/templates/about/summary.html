mListy allows you to create custom named Lists with a cover of your choice. After that you can search for a Movie and add it to the List.
Currently, supports different status types, such as: Completed, Watching or Dropped. You can create a List that is only for Dropped movies, or you can add all types in one list, you have all the freedom to divide them however you want.

mListy search results are provided by TMDB API and Trailers are generated by YOUTUBE API.
mListy is free, doesn't run any ads, and does nothing with your information that you provide. There is zero revenue generated by this site. This is end of course project that was needed to be made.
